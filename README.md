## Getting Started

To run , simply run <code>npm install</code> to install dependencies. The app can then be started by running <code>npm run watch</code> on one terminal and <code>node server.js</code> on another terminal.

### Prerequisites

Node (atleast 8.11.1),
NPM (atleast 5.6.0)
